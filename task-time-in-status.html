Task Time In Status

Be more productive with this app which lets you glance the duration of each ticket in each status


Measure Duration

You can change the unit of measure to minutes, hours or days. We are open for suggestions in format too. Just drop as a message on the desired format to display duration.

Export

Export a duration in current staus report for all board tickets as CSV. A CSV file containing all board tickets along with the durations in current status will be downloaded.

Set Precision

Adjust the precision of you need more accuracy. I.e. if you want to see portions of a day, set unit to days and decimal places to 1 in settings.
