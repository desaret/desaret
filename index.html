Atlassian Jira Add-ons

Issue Time In Status

Easily find the time spent in the current status for the issue being viewed

  • No need for setup as the information is displayed immediately on the right panel along with standard Jira properties
  • If you click over the glance, you can get the time spent on other statuses too

Issue Time In Status Reports

Easily find the time spent in each status for the issues

  • The add-on accumulates the duration for every time the issue has been on a status
  • Durations are stored as custom fields that are indexed and can be used in your reports

Atlassian Trello Power-Ups

Card Time In List

Through this Trello Power-Up you can measure the time a card has been sitting in the current list

  • You can change the unit of measure to minutes, hours or days
  • Adjust the precision of you need more accuracy
  • Export a duration in current list report for all board cards as CSV
  • Change label color based on durance, exclude weekends etc. avalable in the premium version. Drop us a message if interested

Card Age

Another Trello for Gmail

Through the add-on you can create cards or comment to existing cards in Trello from Gmail
